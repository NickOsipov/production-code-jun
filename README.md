Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:NickOsipov/production-code-jun.git
    cd production-code-jun
    ```
2. Create a project:
    ```bash
    uv python pin 3.11
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
